# Avanti Cosmetica - V2 Premium

Sitio web reimaginado con un diseño moderno "Glassmorphism", integrado con WhatsApp y Google Maps.

## Novedades V2
- **Diseño Premium**: Efectos de vidrio esmerilado, gradientes suaves y tipografía "Playfair Display".
- **Contacto Directo**: Botón flotante de WhatsApp en todas las páginas.
- **Ubicación**: Mapa interactivo en la sección de Contacto.
- **Animaciones**: Transiciones suaves y efectos hover avanzados.

## Instalación y Ejecución

Al igual que antes, necesitarás instalar las dependencias una sola vez:

```bash
npm install
```

### Para ver el sitio (Desarrollo)
```bash
npm run dev
```
Visita `http://localhost:3000`

### Para despliegue final
```bash
npm run build:css
npm start
```
